## Development
+ Instal Go Versi 1.8.3.
+ Git clone `git clone https://github.com/harunnryd/cuit-nanimus.git`
+ Instal dependency project : `go get`
+ Jalankan development mode : `go run *.go`
+ Open browser in : `localhost:8080/`
+ Cobain di POSTMAN
